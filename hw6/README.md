# HW6

Put the course data file at `hw6/future_data.zip` first. The data file is not tracked by Git because it is larger than GitHub's normal single-file limit.

Run the experiment from the repository root:

```bash
/Users/quanchen/miniconda3/envs/quant/bin/python hw6/run_hw6.py
```

The script reads `future_data.zip` with pickle `rb` mode, calculates technical indicators and filtering signals, and writes figures/tables under `hw6/outputs`.

The written analysis is in `report.md`; it is maintained manually rather than generated by Python.
